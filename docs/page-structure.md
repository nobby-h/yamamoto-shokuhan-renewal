## トップページ セクション構成案

参考サイトのデザイン要素を統合した構成:

### 1. ヒーローセクション
- **実装**: Swiper.js フェードスライダー + 動画背景オプション
- **参考**: 大橋運輸（Swiper）、和光興業（背景画像フェード）
- キャッチコピー + サブテキスト
- CTAボタン（お問い合わせ / 採用情報）

### 2. ビジョン・理念セクション
- **実装**: パララックス背景 + テキストフェードイン
- **参考**: 夢前急送（背景固定スクロール）
- 企業メッセージ、代表写真

### 3. サービス紹介セクション
- **実装**: カードグリッド（3列）+ ホバーエフェクト
- **参考**: 桐原商事（カード型レイアウト）
- 各サービスのアイコン/イラスト + 概要テキスト

### 4. 数字で見るヤマモト食販
- **実装**: カウントアップアニメーション + アイコン
- 創業年数、従業員数、配送台数、取引先数 等

### 5. トラックアニメーションセクション
- **実装**: SVGトラックの画面横断CSS/GSAPアニメーション
- **参考**: ヨシダ商事（トラック走行アニメ）、桐原商事（横断アニメ）
- セクション間のアクセント・遊び心

### 6. ニュース・お知らせ
- **実装**: 最新3〜5件表示 + 「一覧へ」リンク
- 日付 + カテゴリバッジ + タイトル

### 7. 採用情報CTA
- **実装**: 全幅背景画像 + オーバーレイ + 大型CTA
- 「一緒に働きませんか？」メッセージ
- 外部求人サイトへのリンクも併記

### 8. お問い合わせセクション
- **実装**: 電話番号 + フォームリンク + MAP
- 営業時間、アクセス情報

### 9. フッター
- サイトマップ、会社情報、グループリンク、SNS
