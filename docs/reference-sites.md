# 参考サイト分析

## クライアント指定の参考サイト

### 1. ヨシダ商事運輸（兵庫県西宮市）
- **URL**: https://www.yoshidashoji.co.jp/
- **気に入ったポイント**: イラスト、アニメーションなど全体的なイメージ
- **デザイン特徴**:
  - CSSキーフレームアニメーション（トラック走行、タイヤ回転が無限ループ）
  - イラスト要素をposition: fixed/absoluteで多層配置し視差効果
  - カラー: 青(#207ec3, #368ac9)、緑(#01b700)、ピンク(#cf2f93)
  - 右側固定ドロワーナビゲーション（120px幅）
  - jQuery 1.8.3 + CSS3 keyframes + Flexbox
- **取り入れるべき要素**:
  - トラック・配送イラストのSVGアニメーション
  - 背景の多層レイヤー効果
  - 業種を象徴するイラストの積極活用

### 2. 夢前急送株式会社（姫路市）
- **URL**: http://www.yumekyu48.co.jp/
- **気に入ったポイント**: 背景が固定されて全面コンテンツがスライドする
- **デザイン特徴**: ※SSL証明書エラーによりアクセス不可
- **取り入れるべき要素**:
  - パララックススクロール（background-attachment: fixed）
  - セクション単位の全画面スライド効果
  - ScrollTriggerによるセクション遷移アニメーション

### 3. 桐原商事（会社案内ページ）
- **URL**: https://kirihara-transport.co.jp/company_profile/
- **気に入ったポイント**: 色遣いやアニメーション、見出しが見やすい
- **デザイン特徴**:
  - カラー: 青緑(#04a6d5, #19c9fa)を主軸にした信頼感ある配色
  - トラックイラストの横断アニメーション（moveimg: 20秒ループ）
  - カード型レイアウトで情報整理
  - 雲イラストを背景装飾に使い親しみやすさ演出
  - WordPress + jQuery + Drawer.js + Fancybox
- **取り入れるべき要素**:
  - 青系メインカラーの清潔感ある配色
  - セクション見出しのイラスト付きグループ分け
  - 明確な情報階層構造

### 4. 大橋運輸（瀬戸市）
- **URL**: https://www.0084.co.jp/
- **気に入ったポイント**: 色使い、スライド、見出しの大きさと色
- **デザイン特徴**:
  - 白/グレー基調 + 緑アクセント
  - Swiperカルーセル（5秒間隔フェード切替、PC/SP画像切替対応）
  - SVG形式の見出し、階層的なサイズ設定
  - 固定ヘッダー + パンくずリスト
  - jQuery + Swiper + matchHeight.js + Lazy load + Google Fonts
  - Schema.org構造化データ（JSON-LD）
- **取り入れるべき要素**:
  - Swiperベースのヒーロースライダー
  - 見出しの大胆なサイズ設定
  - 遅延読み込み（Lazy load）の採用
  - 構造化データの実装

### 5. 和光興業（宮城・福島）
- **URL**: https://wako-kogyo.jp/
- **気に入ったポイント**: 動画や写真の差込、サッパリとした配置がシンプルで見やすい
- **デザイン特徴**:
  - ヒーロー領域で背景画像を3秒フェード自動切替
  - YouTube動画リンク設置
  - 余白を活かした読みやすいレイアウト
  - ナビ→ヒーロー→ビジョン→サービス→フッターの明快な構成
  - jQuery + bxSlider + backstretch
  - モバイル対応ハンバーガーメニュー
- **取り入れるべき要素**:
  - ヒーロー動画/スライドショー
  - 余白を活かしたミニマルレイアウト
  - 明快なセクション構成
